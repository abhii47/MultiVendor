# MultiVendor Project 🌐

## Tech Stack 🛠️
- TypeScript 🔷
- Node.js 🚀
- MySQL 💾
- JWT 🔐
- Stripe 💳
- AWS S3 ☁️
- SendGrid 📧
- Winston 📊

## Project Structure 📁
- `/src`
  - Contains all the source code.
- `/config`
  - Configuration files.
- `/tests`
  - Unit and integration tests.

## Coding Standards 📝
- Use ES6 syntax.
- Consistent indentation (4 spaces).
- Descriptive variable and function names.
